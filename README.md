# CUT&Tag Peak Calling Analysis

## 📌 Overview
This project focuses on analyzing CUT&Tag data to identify chromatin enrichment regions using peak-calling tools. The goal is to evaluate different computational approaches and compare their ability to detect biologically meaningful signals.

## 🎯 Objectives
- Perform peak calling on CUT&Tag datasets
- Compare tools (e.g., MACS2, HOMER, etc.)
- Identify significant genomic regions
- Evaluate differences in peak detection and annotation

## 🧪 Methods
- Data preprocessing and quality control
- Alignment to reference genome
- Peak calling
- Downstream analysis (annotation, visualization)
